# Conversor de Distancia

## Descrição

Este laboratório tem o objetivo de containerizar uma aplicação me Python, essa aplicação faz a conversão de km para metro ou vice versa.

## Tecnologia utilizadas
* Codigo fonte da python
* Editor de texto
* docker

## Instalação

* Docker <https://docs.docker.com/engine/install/ubuntu/>
* Editor de texto - VsCode <https://code.visualstudio.com/docs/setup/linux>
* Faça o fork para obter o codigo fonte
* Construa o Dockerfile para criar a imagem
